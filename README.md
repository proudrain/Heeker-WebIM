# HEEKER (Web IM)

## 功能简介
+ 注册登录。(注册提供邀请码机制)
+ 添加好友。
+ 在线聊天。
+ 离线消息。

## 技术实现
+ 前端使用**Angularjs**框架。
+ 后端使用python的**tornado**框架。
+ 核心通信使用**WebSocket**。

## 依赖
+ python-3.4
+ tornado-4.1
+ pymysql-0.6.3

## Tips
+ `python/main_service.py`为后端主程序。
+ 数据库只需要执行该`sql/heeker.sql` SQL语句即可导入全部测试用数据。
+ 可通过`python/invite_keys.py`查看现有邀请码。
